Portofolio Website - Teguh Esa Maulanna

1. Deskripsi Singkat
Website ini dibuat menggunakan HTML, CSS, JS, dan Bootstrap 5. Website ini berisikan informasi
seperti profil singkat saya, keahlian saya, project yang saya kerjakan, galeri project dan photography,
informasi kontak, dan form kontak. Setiap halaman yang dibuat sudah full responsive sehingga tetap nyaman
bagi pengguna mobile. Setiap halaman juga sudah dilengkapi dengan navbar untuk navigasi setiap halaman dan 
footer.

2. Fitur
Halaman home:
- profil singkat.
- card skill.
- carousel galeri photography. 
Halaman about:
- profil pembuat.
- accordion berisi perjalanan singkat pembuat.
Halaman portofolio:
- filter tabs untuk mempermudah pencarian project berdasarkan kategori badge
- galeri project yang pernah dibuat, setiap projcect ditampilkan dengan card, modal, carousel, badge,
dan deskripsi singkat setiap project.
Halaman contact:
- contact form untuk pengunjung dapat memberikan pertanyaan, feedback, penawaran project, dan kerjasama.
- informasi contact pembuat.

3. Teknologi yang Digunakan
- HTML: Untuk struktur konten website
- CSS: Untuk styling dan desain visual
- JavaScript: script untuk portfolio filter dan form validation
- Bootstrap: sebagai framework CSS
- Bootstrap Icons: Untuk icon pada footer dan tombol
